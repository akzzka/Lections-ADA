git init - создает git репозитории из текущей папки (пишется один раз в самом начале)
git add . - (Либо место точки любой файл из директории) - добавляет файлы и папки для отслеживания в index
(эту команду пием раз перед командой commit)
git commit -m 'Сообщение' - создает новую версию, если были изменения в отслеживающихся файлах

Полный путь от создания локального репозитория, до отправки на удаленный
1) создаем новый репозитории на Github
2) git init - инциализируем локальный репозиторий, после этой команды в директории должен появиться 
файл .git, для его просмротра можео выаолнить ls -a
3) git add - добавляем все файлы/папки в индекс git
4) git commit -m 'comment'
5) устанавливаем связь через SSH - ключ
    1) генерируем ssh - ключ через команду ssh-keyen
