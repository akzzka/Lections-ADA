Директория - это обычная папка

ctr + alt + t = команда для запуска терминала (UBUNTU)

sudo = это команда для передоставления прав суперпользавателя

ls = команда для вывода всех файлов/папок в текущей директории
ls -a = команда которая показывает все и скрытые файлы/папки
ls -i = команда которая выводит более детальную информацию о файлах/папках
cd test (change directory) = команда для перехода в другую папку
cd = команда для перехода в домашную папку
cd .. = команда для перехода на 1 папку выше (назад)
cd ../.. =  команда для перехода на 2 папки выше (назад)
mkdir = команда для создани директория 
mkdir test test2 = команда создаст 2 папки, в текущей директории
mkdir test/test2 = команда создаст папку в существующей папке

touch test.txt = команда для создания файла 
rm test.ry = команда для удаления файла
rm -rf test.ry test = команда для удаления файла/директории
rmdir <название папки> = команда для удаления пустой папки

clear/ctr + l = команда/горячая клавиша для очистки терминала

mv test1.py test/ - перемещает файл test1.txt в папку test (обязательно указываем / в конце названия папки)
mv test1.txt hello.txt - переименовывает файл test1.txt в hello.txt
mv test/ test2/ - переименовывает папку test/ на test2/

history - показывает все команды, которые были использованы в этом терминале
history 10 - показывает последние 10 команд
history | grep <какое-то слово> - показывает все команды с использованием слова в скобках
    например:
history | grep test

pwd (print working directory) - показывает абсолютный путь до текущей директории (начиная с корневой папки /)

cat <название файла> (например: cat test.txt) - показывает содержимое файла 

nano test.txt - открывает файл test.txt в редакторе nano, если такого файла нет, то создаст его (в случае если файл не пустой)

сохранения файла, и выход из редактора nano
ctrl + x, y, ENTER
        |
ctrl + s, ctrl + 

code test(code test.txt) - открывает в редакторе кода vs-code указанную папку, или файл
code . - открывает vs-code в текущей директории


apt - системный установщик пакетов
snap - установщик пакетов(но чаще всего установщик приложений)
